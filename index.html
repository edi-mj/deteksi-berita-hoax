Tentang Hoax Buster

Hoax Buster merupakan proyek berbasis machine learning yang dikembangkan untuk membantu mendeteksi berita hoaks berbahasa Indonesia secara otomatis. Sistem ini memanfaatkan teknologi Natural Language Processing (NLP) untuk menganalisis karakteristik teks pada berita digital.

Dengan mempelajari pola linguistik dan distribusi kata dalam berita, Hoax Buster bertujuan menyediakan alat verifikasi awal yang cepat dan mudah diakses, sehingga pengguna dapat menyaring informasi sebelum membagikannya di ruang digital.

Metodologi & Dataset

Model machine learning pada sistem ini dilatih menggunakan dataset berita berbahasa Indonesia yang telah dilabeli ke dalam kategori hoaks dan non-hoaks. Proses pelatihan dilakukan untuk menghasilkan model klasifikasi yang akurat dan stabil.

  • Preprocessing: Pembersihan teks, normalisasi, tokenisasi, penghapusan stopword, dan stemming.
  • Ekstraksi Fitur: Pembobotan kata menggunakan metode TF-IDF (Term Frequency–Inverse Document Frequency).
  • Model: Algoritma Logistic Regression untuk meningkatkan performa klasifikasi.

Analisis Real-time

Sistem memproses input teks secara instan tanpa antrean server, memberikan hasil prediksi dalam hitungan milidetik langsung di peramban Anda.

Akurasi Teruji

Model dilatih secara spesifik untuk mengenali struktur kalimat provokatif, penggunaan huruf kapital berlebih, dan indikator hoax umum lainnya.
